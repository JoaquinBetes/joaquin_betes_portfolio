Aplicación web integral diseñada para la gestión y automatización de las operaciones de un estudio de tatuajes. Proporciona funcionalidades para la administración de turnos, clientes, diseños, tatuadores y políticas del estudio, además de liquidaciones mensuales de sueldos y generación de gráficos e informes de ingresos al estudio.

Frontend

  • Interfaz de usuario: Intuitiva para clientes, tatuadores y encargado.
  • Secciones dinámicas: Visualización de tatuajes/diseños, filtros por categorías y gestión de turnos.
  • Formularios: Registro, reservas y gestión de datos personales.
  • Gráficos interactivos: Informes visuales de ingresos por comisiones mensuales y por tatuador utilizando API Google Charts.

Repositorio: https://github.com/JoaquinBetes/FrontendEstudioTatuaje

Backend

  • Gestión de usuarios: Roles y autenticación (clientes, tatuadores, encargado).
  • Notificaciones: Correo electrónico para confirmaciones y recordatorios.
  • Lógica de liquidación: Automática para los tatuadores.
  • Reglas de negocio: Estados de diseños, cálculo de precios, penalizaciones, etc.

Repositorio: https://github.com/JoaquinBetes/estudioTatuajes
